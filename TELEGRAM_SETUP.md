# 📱 Настройка Telegram Уведомлений

## Самый простой способ получать уведомления о новых заявках!

Telegram уведомления намного проще настроить, чем email. Вам не нужен пароль от почты!

## Шаг 1: Создайте Telegram бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для бота (например: "Лиора Продленка")
   - Придумайте username для бота (должен заканчиваться на `bot`, например: `liora_prodlenka_bot`)
4. **Скопируйте токен**, который даст вам BotFather
   - Выглядит примерно так: `8590033940:AAHiwDQienOKIOT9gJfx_iJhykEEOQJ3Fis`

## Шаг 2: Получите свой Chat ID

1. Откройте Telegram и найдите **@userinfobot**
2. Отправьте команду `/start`
3. Бот покажет ваш Chat ID (это число, например: `6469030723`)
4. **Скопируйте это число**

## Шаг 3: Настройте config.js

1. Скопируйте пример конфигурации:
   ```cmd
   copy config.example.js config.js
   ```

2. Откройте `config.js` и заполните:
   ```javascript
   telegram: {
       botToken: '123456789:ABCdefGHIjklMNOpqrsTUVwxyz',  // Токен от BotFather
       chatId: '123456789'                                 // Ваш Chat ID
   }
   ```

3. Сохраните файл

## Шаг 4: Перезапустите сервер

После настройки перезапустите сервер. В консоли должно появиться:
```
📱 Telegram notifications: ENABLED
```

## Готово! 🎉

Теперь при каждой новой заявке вы будете получать уведомление в Telegram!

## Что будет в уведомлении:

- 🔔 Имя родителя
- 📞 Телефон (можно сразу позвонить)
- 👶 Информация о ребенке
- ⏰ Удобное время для звонка
- 📅 Дата и время заявки

## Альтернатива: Админ-панель на сайте

Если не хотите настраивать Telegram, просто откройте в браузере:
```
http://localhost:3000/admin.html
```

Там вы увидите все заявки в реальном времени с возможностью автообновления!

## Решение проблем

**Бот не отвечает / не отправляет сообщения:**
- Проверьте правильность токена
- Проверьте правильность Chat ID
- Убедитесь, что вы отправили `/start` боту @userinfobot

**Ошибка "chat not found":**
- Убедитесь, что Chat ID правильный
- Попробуйте отправить боту любое сообщение перед настройкой
