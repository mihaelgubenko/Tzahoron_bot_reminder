# Лендинг "Продленка в детском саду Лиора"

Одностраничный сайт для тестирования спроса на услугу продленки в детском саду "Лиора".

## 🚀 Быстрый старт

### Требования
- Node.js (версия 14 или выше)
- npm (устанавливается вместе с Node.js)
- Email аккаунт для уведомлений (Gmail, Outlook или другой SMTP)

### Настройка уведомлений (ВАЖНО!)

**У вас есть 3 способа получать уведомления о новых заявках:**

#### 🎯 Способ 1: Админ-панель на сайте (САМЫЙ ПРОСТОЙ!)

Просто откройте в браузере:
```
http://localhost:3000/admin.html
```

Там вы увидите все заявки в реальном времени с:
- Статистикой (всего, сегодня, за неделю)
- Автообновлением каждые 30 секунд
- Возможностью экспорта в CSV
- Кликабельными номерами телефонов

**Не требует никакой настройки!** Просто откройте страницу.

#### 📱 Способ 2: Telegram уведомления (РЕКОМЕНДУЕТСЯ!)

Намного проще, чем email! Не нужен пароль от почты.

**Быстрая настройка:**
1. Создайте бота через @BotFather в Telegram
2. Получите Chat ID через @userinfobot
3. Заполните настройки в `config.js`

📖 **Подробная инструкция:** см. `TELEGRAM_SETUP.md`

#### 📧 Способ 3: Email уведомления (опционально)

Если хотите получать на email:
1. Скопируйте `config.example.js` в `config.js`
2. Настройте SMTP (Gmail, Outlook и т.д.)
3. Укажите email администратора

📖 **Подробная инструкция:** см. `EMAIL_SETUP.md`

**Рекомендация:** Используйте админ-панель или Telegram - это проще и удобнее!

### Установка и запуск

#### Вариант 1: Автоматический запуск (рекомендуется)
1. Дважды кликните на файл `start-server.bat`
2. Сервер запустится автоматически

#### Вариант 2: Ручной запуск через командную строку

**Windows CMD:**
```cmd
npm install
node server.js
```

**Git Bash:**
```bash
npm install
node server.js
```

### Доступ к сайту
После запуска сервера откройте в браузере:
- **Главная страница:** http://localhost:3000
- **Админ-панель (все заявки):** http://localhost:3000/admin.html
- **Просмотр заявок (CSV):** http://localhost:3000/api/leads

## 📝 Тестирование формы

### Как протестировать:

1. **Запустите сервер** (см. выше)

2. **Откройте сайт** в браузере: http://localhost:3000

3. **Заполните форму:**
   - Имя родителя (обязательно)
   - Номер телефона (обязательно)
   - Имя и возраст ребенка (необязательно)
   - Удобное время для звонка
   - Согласие на обработку данных (обязательно)

4. **Отправьте форму** - нажмите кнопку "Отправить заявку"

5. **Проверьте результат:**
   - На странице появится сообщение об успешной отправке
   - Данные сохранятся в папке `leads/`
   - Каждая заявка сохраняется как отдельный JSON файл
   - Все заявки также записываются в `leads/leads.csv`
   - **Если настроен email:** администратор получит уведомление на почту
   - В консоли сервера появится сообщение о новой заявке

### Просмотр заявок

**Способ 1: Через браузер**
- Откройте: http://localhost:3000/api/leads
- Увидите CSV файл со всеми заявками

**Способ 2: Через файловую систему**
- Откройте папку `leads/`
- Просмотрите JSON файлы с отдельными заявками
- Откройте `leads.csv` в Excel или текстовом редакторе

## 🔧 Настройка

### Изменение порта сервера
Откройте `server.js` и измените:
```javascript
const PORT = 3000; // Измените на нужный порт
```

### Настройка сохранения данных
Данные сохраняются в папке `leads/`. Для изменения формата или места сохранения отредактируйте `server.js`.

## 🐛 Решение проблем

### Порт уже занят
Если порт 3000 занят, измените PORT в `server.js` на другой (например, 3001, 8080).

### Ошибка "npm не найден"
Установите Node.js с официального сайта: https://nodejs.org/

### Форма не отправляется
1. Убедитесь, что сервер запущен
2. Откройте консоль браузера (F12) и проверьте ошибки
3. Проверьте, что все обязательные поля заполнены

## 📧 Обработка заявок

Все заявки автоматически сохраняются в:
- **JSON формат:** `leads/lead_TIMESTAMP_NAME.json` - для программной обработки
- **CSV формат:** `leads/leads.csv` - для просмотра в Excel

Каждая заявка содержит:
- Дата и время отправки
- Имя родителя
- Номер телефона
- Информация о ребенке
- Удобное время для звонка
