# 🚀 Установка и запуск

## Быстрый старт

### Шаг 1: Установите зависимости

**Вариант A: Через командную строку CMD (рекомендуется)**
```cmd
npm install
```

**Вариант B: Через Git Bash**
```bash
npm install
```

**Вариант C: Автоматически через start-server.bat**
- Просто запустите `start-server.bat` - он сам установит зависимости при первом запуске

### Шаг 2: Настройте Telegram (опционально, но рекомендуется)

1. Откройте `config.js`
2. Убедитесь, что там указаны:
   - `botToken` - токен от @BotFather
   - `chatId` - ваш Chat ID от @userinfobot

### Шаг 3: Запустите сервер

**Через .bat файл:**
- Дважды кликните на `start-server.bat`

**Или через командную строку:**
```cmd
node server.js
```

### Шаг 4: Откройте сайт

- **Главная страница:** http://localhost:3000
- **Админ-панель:** http://localhost:3000/admin.html

## Тестирование

1. **Откройте главную страницу** и заполните форму
2. **Проверьте админ-панель** - должна появиться новая заявка
3. **Проверьте Telegram** - должно прийти уведомление (если настроено)

## Решение проблем

**Ошибка "Cannot find module 'nodemailer'"**
- Запустите `npm install` в командной строке CMD (не PowerShell)
- Или используйте `start-server.bat` - он сам установит зависимости

**PowerShell блокирует npm**
- Используйте обычную командную строку CMD вместо PowerShell
- Или запустите PowerShell от имени администратора и выполните: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

**Сервер не запускается**
- Убедитесь, что Node.js установлен: `node --version`
- Убедитесь, что порт 3000 свободен
- Проверьте, что все зависимости установлены: `npm install`
