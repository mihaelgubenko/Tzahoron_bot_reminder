# Изображения

## Hero изображение

Поместите изображение детей на игровой площадке в этот файл:
- **Имя файла:** `hero-children.jpg` (или `.png`)
- **Рекомендуемый размер:** 1200x1000px или больше
- **Формат:** JPG или PNG

Изображение будет автоматически отображаться в hero секции главной страницы.

## Альтернатива

Если у вас есть изображение с другим именем, измените путь в `index.html`:
```html
<img src="images/ваше-изображение.jpg" alt="..." class="hero-image-img">
```
