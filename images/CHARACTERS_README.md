# Изображения персонажей Муми-троллей

Для использования настоящих персонажей из мультфильма, разместите изображения в этой папке с следующими именами:

## Проблемы (Problems):
- `moomin-clock.png` - Муми-тролль с часами (работаете сменами)
- `moomin-papa.png` - Муми-папа (утомительно возить)
- `moomin-friends.png` - Два Муми-тролля (не хватает общения)
- `sniff.png` - Снифф (няня дорого)

## Функции (Features):
- `moomin-mama-food.png` - Муми-мама с едой (горячий обед)
- `moomin-sleep.png` - Муми-тролль спит (дневной сон)
- `moomin-mama.png` - Муми-мама (присмотр педагогов)
- `little-my.png` - Малышка Мю (развивающие игры)
- `moomin-clock-flex.png` - Муми-тролль с часами (гибкий график)

## Преимущества (Advantages):
- `moomin-parents.png` - Муми-мама и Муми-папа вместе (опытные педагоги)
- `snufkin.png` - Снусмумрик (безопасность)
- `sniff-doc.png` - Снифф с документом (прозрачность)
- `little-my-location.png` - Малышка Мю с геолокацией (удобство)

## Формат изображений:
- Рекомендуемый формат: PNG с прозрачным фоном
- Размер: примерно 200x200px или больше (изображения будут автоматически масштабироваться)
- Стиль: аниме-версия Муми-троллей

## Важно:
Если изображения не найдены, будут использоваться SVG-заглушки в стиле Муми-троллей.

**Примечание об авторских правах:** Использование изображений персонажей Муми-троллей может требовать разрешения правообладателей. Для тестового сайта это допустимо, для коммерческого использования рекомендуется получить лицензию.
